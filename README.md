# Prometheus

Prometheus is an interactive browser-based space simulation that lets users explore planets, stars, constellations, and deep-space data in a 3D scene.

## Live Demo

[View Prometheus](https://nikolairangl.github.io/Prometheus-/)

## Features

- Interactive 3D solar system and star field
- Camera controls for orbiting, zooming, and free exploration
- Speed up time to watch orbital motion unfold faster
- Rewind time to view celestial positions in the past
- Visit distant stars and explore beyond the solar system
- Planet labels and quick travel controls
- Realistic planet texture support
- Star catalog and constellation data
- Runs directly in the browser with no backend required

## Tech Stack

- JavaScript
- HTML
- CSS
- Three.js
- GitHub Pages

## Run Locally

The project uses ES modules, so open it through a local server instead of double-clicking `index.html`.

```bash
npx serve .
```

Then open the local URL shown in the terminal.

You can also use Cursor or VS Code with the Live Server extension: right-click `index.html`, then choose **Open with Live Server**.
